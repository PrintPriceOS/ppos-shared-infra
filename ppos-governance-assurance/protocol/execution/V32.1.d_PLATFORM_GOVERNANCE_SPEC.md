# V32.1.d Platform & Governance Spec v1.0.0

## 1. Overview
The **Platform & Governance Spec (V32.1.d)** defines the second wave of foundation repositories. These repos provide the execution engine (`core-platform`) and the trust/compliance layer (`governance-assurance`) of PrintPrice OS.

## 2. Target Repositories
- `ppos-core-platform`: Kernel, state management, and runtime foundation.
- `ppos-governance-assurance`: Release governance, audit evidence, and trust policy enforcement.

## 3. Core Packages
### `ppos-core-platform`
- `kernel-state`: Core state management and transition logic.
- `runtime-foundation`: Service discovery and composition primitives.

### `ppos-governance-assurance`
- `release-governance`: Governance gates and policy resolution.
- `trust-assurance`: Evidence handling and audit trace management.

## 4. Integration Strategy
- Consumes `@ppos/*-contracts` from the shared repo.
- Enforces protocol compliance via `ppos-federation-protocol` contracts.
- Provides implementation hooks for strategic domain logic.
