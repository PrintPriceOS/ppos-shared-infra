# V32.1.b Bootstrap Content Spec v1.0.0

## 1. Introduction
The **Bootstrap Content Spec (V32.1.b)** defines the actual source code and manifest content for the first two foundation repositories: `ppos-shared-contracts` and `ppos-federation-protocol`.

## 2. Source Repositories
- `ppos-shared-contracts`: Authoritative JSON schemas and TypeScript types.
- `ppos-federation-protocol`: Core FEP protocol implementation and routing logic.

## 3. Key Components
- **Root Manifests**: package.json, pnpm-workspace.yaml, tsconfig.base.json.
- **Contract Packages**: `federation-contracts`, `governance-contracts`.
- **Protocol Packages**: `fep-protocol`, `fep-validation`.

## 4. Integration Rules
- `ppos-federation-protocol` MUST consume `@ppos/*` packages from `ppos-shared-contracts` via `workspace:*` dependencies.
- No business logic in `ppos-shared-contracts`.
- No authoritative schema duplication in `ppos-federation-protocol`.
