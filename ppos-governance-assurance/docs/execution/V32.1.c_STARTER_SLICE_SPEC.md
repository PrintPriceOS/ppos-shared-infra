# V32.1.c Starter Slice Spec v1.0.0

## 1. Overview
The **Starter Slice Spec (V32.1.c)** formalizes the physical file contents for the foundational "slice" of PrintPrice OS repositories. This slice provides a functional, contract-first starting point for implementation.

## 2. Included Repositories
- `ppos-shared-contracts`: authoratitative schemas and types.
- `ppos-federation-protocol`: core FEP implementation.

## 3. Structural Rules
- **Contract-First**: Every package should have a standard `src/contracts`, `src/schemas`, and `src/types` structure.
- **Root Uniformity**: All repos share the same root config files (editorconfig, lint, workspace).
- **Service Decomposition**: Logic is separated into `services/` and `resolvers/`.

## 4. Content Integrity
- All JSON schemas must compile with Ajv.
- All TypeScript types must match their corresponding JSON schemas.
- Cross-repo dependencies are managed via `workspace:*` and tsconfig paths.
