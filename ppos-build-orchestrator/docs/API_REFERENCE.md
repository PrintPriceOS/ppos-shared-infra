# API Reference

## Core Packages
- `@ppos/build-program-core`: State enums and base contracts.
- `@ppos/workspace-bootstrapper`: `bootstrapWorkspace(config)`.
- `@ppos/repo-provisioner`: `provisionRepository(repoId, template)`.
- `@ppos/gate-evaluator`: `evaluateStoryGate(story, gate)`.

## CLI Commands
- `resolve`: Build a program plan.
- `provision`: Apply templates to repos.
- `readiness`: Generate readiness reports.
- `promote`: Approve program for promotion.
